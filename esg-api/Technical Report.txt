## Data Sources

### ESG and Financial Data
- **FinancialModelingPrep API** (https://site.financialmodelingprep.com)
  - Used to fetch ESG scores, carbon intensity, and controversy scores
  - Free tier allows 250 requests/day
  - Data ingested via `scripts/data_ingestion.py` and stored in PostgreSQL

- **Yahoo Finance API** (via yfinance Python library)
  - Used for historical stock prices and financial metrics (P/E, EPS)
  - Data cached locally to reduce API calls

### Data Processing
- Initial dataset: 50 companies across 5 sectors (Technology, Finance, Energy, Healthcare, Consumer)
- Data via scheduled script
- All data appropriately licensed for academic use per API terms of service
